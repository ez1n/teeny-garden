# Teeny Garden

| 당신의 일상을 빛내줄 작은 정원, 티니 가든

<br/>

## 소개

```
반지, 목걸이, 키링, 휴대폰 스트랩, 그립톡 등을 판매하는 악세사리 쇼핑몰입니다.
```

<br/>

## 프로젝트 실행

```Bash
yarn install
```

```Bash
yarn start
```

<br/>

## 기술 스택

- `Javascript`
- `React`
- `tailwind css`
- `react-query`
- `react-router`
- `firebase`
- `cloudinary`
